This website is created to present interesting information about tourism in central Thailand, gathering important tourist attractions, local specialities, and sacred places for worship. It was developed as a part of the Full Stack Web Development course (Assignment 5).

## 🗂 Website Structure

This website consists of 4 main pages:

1. **🏠 Home (`index.html`)**
   - Welcome Page with banners and image sliders
   - General information and geographical characteristics of the central region
   - Map and website overview for convenient navigation

2. **⛰️ Tourist Attractions (`one.html`)**
   - A collection of 10 recommended tourist attractions in central Thailand (e.g., Koh Kret, Khun Dan Prakarn Chon Dam, Amphawa Floating Market, etc.)
   - Includes details, links for more information, and map coordinates for each location

3. **🍜 Local Specialties (`two.html`)**
   - Presents 10 famous local products and foods of central provinces (e.g., Khanom Krok, Bang Bo Gourami, Tom Yum Goong, Dragon Jars, etc.)

4. **🙏 Sacred Places (`three.html`)**
   - Recommends 10 sacred places for paying respects and asking for blessings (e.g., Wat Phra Kaew, Luang Phor Sothon, Phra Pathommachedi, Erawan Shrine)

---

## 🛠️ Tech Stack

This website is developed using basic web technologies and tailored to support usage across all screens (Responsive Design):

- **HTML5**: Main website structure
- **CSS3**: Beautiful decoration and layout formatting (Custom Styling & Animations)
- **Bootstrap 5.3.3**: Framework that helps manage Responsive Design and UI Components (e.g., Navbar, Card, Grid system)
- **Google Fonts**: Uses the **"Prompt"** font for a beautiful, modern, and readable appearance

---

## 🚀 Installation & Usage

1. **Clone or download** all project files to your computer
2. Make sure the `images/` folder and all accompanying images are complete
3. **Open the file** `index.html` with any web browser (e.g., Google Chrome, Microsoft Edge, Safari) to start using and browsing the website
